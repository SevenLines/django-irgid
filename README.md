irgid
=====

[![Build Status](https://travis-ci.org/SevenLines/django-irgid.svg?branch=master)](https://travis-ci.org/SevenLines/django-irgid)

Static
-----------

```
yarn watch
```


to build

```
yarn build
```

then

```
./manage.py assets build
```



Build translates
---------------

Update translates

```
./manage.py makemessages -l en   
```


compile messages

```
./manage.py compilemessages -l en
```
